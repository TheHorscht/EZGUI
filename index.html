EZGUI v0.2.0 Reference

Elements

DOMElement (All Elements)

Every element is of this type and holds some common functionality.

Attributes

forEach

[ loop expression ]
Allows you to repeat the rendering of an element for each item in a table. Example: 
local data_context = {
  fruit_basket = { "Apple", "Banana", "Tomato" }
}

<Text forEach="fruit in fruit_basket">{{ fruit }}</Text>
Result: 
<Text>Apple</Text>
<Text>Banana</Text>
<Text>Tomato</Text>

CSS Properties

align_items_horizontal

[ left | center | right ]
Specifies horizontal alignment of items in a Layout when direction is vertical.

align_items_vertical

[ top | center | bottom ]
Specifies vertical alignment of items in a Layout when direction is horizontal.

align_self_horizontal

[ left | center | right ]
Similar to the 'text-align' (real) CSS property. Only works when width is being set and is greater than the content width.

align_self_vertical

[ top | center | bottom ]
Similar to the 'text-align' (real) CSS property, but vertical. Only works when height is being set and is greater than the content height.

border

[ boolean ]
Renders a border around an element.

color

[ RGB(A) Hex Color #FFFFFF(FF) ]
Colors an element, does not work properly with Image, especially alpha does not work at all on Image.
